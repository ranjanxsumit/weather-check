Features

- Fetch current weather (Open-Meteo current_weather)
- Responsive liquid glass UI using Tailwind and CSS backdrop-filter

Quick start
1. Install dependencies

```powershell
npm install
```

2. Run development server

```powershell
npm run dev
```

3. Build for production

```powershell
npm run build
npm run preview
```.
